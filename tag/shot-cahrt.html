Creating Grantland style shot charts

Date Sun 07 May 2017 Category Posts Tags python / NBA / shot cahrt

In this post I will show how to make Grantland type shot charts with Python. The basic idea of the shot chart is: * Shots are represented by hexagons * More shots = larger hexagon and vice versa * Color of the hexagon represents the FG % relative to the league average. The FG % is done in zones not per spot (due to lack of statistics in each spot)}
